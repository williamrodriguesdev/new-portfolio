Habilidades

Html

HTML é a linguagem usada para estruturar o conteúdo de sites, organizando elementos de forma semântica o HTML melhora a navegação e torna os sites mais acessíveis garantindo que pessoas com diferentes capacidades possam acessar o conteúdo.

CSS

É a linguagem utilizada para estilizar e formatar o conteúdo de páginas web. Com ela, você pode controlar aspectos como cores, fontes, espaçamentos e layouts. Além disso, CSS oferece recursos avançados, como responsividade e animações, garantindo que os sites se adaptem a diferentes dispositivos e tamanhos de tela.

JavaScript

JavaScript é a linguagem usada para adicionar interatividade e funcionalidade a sites. Permite criar elementos dinâmicos, tornando a navegação mais fluida e interativa, melhorando a experiência do usuário.

React

É uma biblioteca JavaScript usada para criar interfaces dinâmicas, facilita o desenvolvimento de sites ao usar componentes reutilizáveis, que permitem organizar e otimizar o código. Além disso, React atualiza apenas o necessário na tela, melhorando a performance do site.

Git

É um sistema de controle de versão que permite rastrear alterações em arquivos e colaborar em projetos de forma eficiente. Ele facilita o trabalho em equipe, permitindo que vários desenvolvedores trabalhem simultaneamente no mesmo código sem conflitos.

GitHub

GitHub é uma plataforma de hospedagem de código que utiliza o Git para controle de versão. Ela permite que desenvolvedores armazenem, compartilhem e colaborem em projetos de software. Com recursos como repositórios, pull requests e issues, GitHub facilita a colaboração em equipe e a gestão de projetos
